How to use:

Sorry for the text dump but you should know that

All fields are editable.

EXCEPT FOR the remaining points field which is calculated from all numerical values of the six main stats (Grade Level, Pokethulu Lore, Sanity, Trash Talking, Shoplifting, and Phys Ed).

All fields main fields can be randomized

EXCEPT FOR Remaining Points, Name, and Hometown

The randomize button is the green button with a dice logo.

The proper order to randomize each stat(IF YOU SO CHOOSE) is:

1: age : limits the range of a cultist's grade level

2: grade level and pokethulu lore : grade level is a maximum number this is 4 below the age. i.e 16 y/o - 4 = grade level 12

3: sanity: takes into account either the cultists's grade level or pokethulu lore value. Where one of the values and the sanity value cannot exceed 13.

4: Trash Talking, Shoplifting, Phys Ed, Cultist Lucky Stat.

All values will fall into the range of 1 to 12 when saved with blue button at the top that has a floppy disc logo.

This cultist creation tool is not perfect and may allow the creation of a broken character.

However it should help in assigning values and give a general overview for creating a unique cultist
